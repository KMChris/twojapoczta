# Dokumentacja TwojaPoczta

Cała wiedza o projekcie w pięciu dokumentach:

| Dokument | Dla kogo | O czym |
| -------- | -------- | ------ |
| [**Wdrożenie krok po kroku**](wdrozenie.md) | administrator | Od świeżego VPS-a do działającej poczty z HTTPS, odbiorem ze świata i DKIM. ~30 minut. |
| [**Konfiguracja**](konfiguracja.md) | administrator | Wszystkie zmienne środowiskowe, typowe zestawy, rekordy DNS, limity aplikacji. |
| [**Panel administratora**](administracja.md) | administrator | Konta i limity, blokady, DKIM, weryfikacja DNS, catch-all, komunikaty, dziennik zdarzeń. |
| [**Przewodnik użytkownika**](przewodnik.md) | użytkownik | Foldery, pisanie, skróty klawiszowe, paleta poleceń, aliasy, ustawienia. |
| [**Architektura**](architektura.md) | współtwórca | Budowa kodu, model danych, cykl życia wiadomości, API, zasady projektu. |

Nowy na serwerze? Zacznij od [wdrożenia](wdrozenie.md).
Nowy w skrzynce? Zajrzyj do [przewodnika](przewodnik.md).
Chcesz zmieniać kod? Przeczytaj [architekturę](architektura.md).

Projekt powstał w podejściu **zero zależności**. Szczegóły decyzji
projektowych znajdziesz w [architekturze](architektura.md).
